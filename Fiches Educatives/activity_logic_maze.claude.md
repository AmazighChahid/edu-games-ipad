# ACTIVITÉ — Labyrinthe logique

## Objectif pédagogique
Travailler la planification, l’anticipation et la gestion de contraintes.

## Principe Montessori
- Essai-erreur autorisé
- Auto-correction par impossibilité d’action

## Principe du jeu
Atteindre une cible en respectant des contraintes :
- cases interdites
- sens unique
- ordre obligatoire

## UX Flow
1. Vue globale du labyrinthe
2. Tracé du chemin avec le doigt
3. Validation progressive
4. Indices IA si répétition d’erreurs

## UI iPad
- Grille large
- Codes couleur simples
- Bouton reset visible

## IA
"Que se passe-t-il si tu changes de chemin ?"

## Fiche parent
Compétences : planification, persévérance
